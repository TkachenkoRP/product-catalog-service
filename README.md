# Каталог Товаров

## Описание приложения
Консольное приложение для управления каталогом товаров с системой пользователей, возможностью сохранения данных между сеансами.

## Используемые технологии
- **Java 17+** - основной язык программирования
- **Maven** - сборка и управление зависимостями
- **In-Memory хранилище** - быстрый доступ к данным
- **CSV файлы** - хранение данных
- **MapStruct** - маппинг между объектами

## Функциональность

### Управление пользователями
- Регистрация и авторизация
- Управление профилем (имя, email, пароль)

### Управление товарами
- CRUD операции с товарами
- Поиск и фильтрация (по названию, категории, бренду, цене)
- Валидация данных при добавлении

### Справочники
- Управление категориями и брендами
- Защита от удаления используемых категорий/брендов
- Проверка уникальности названий

### Сохранение данных
- Автоматическое сохранение в CSV при выходе
- Загрузка данных при запуске
- Резервное копирование в папку `data/`

## Инструкции по запуску

### Требования
- Java 17 или выше
- Maven 3.6+
