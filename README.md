# Каталог Товаров

## Описание приложения

Консольное приложение для управления каталогом товаров с системой пользователей, возможностью сохранения данных между
сеансами.

## Используемые технологии

* Java 17
* Maven
* MapStruct
* JDBC
* PostgreSQL
* Liquibase
* AssertJ
* Testcontainers

## Функциональность

### Управление пользователями

- Регистрация и авторизация
- Управление профилем (имя, email, пароль)

### Управление товарами

- CRUD операции с товарами
- Поиск и фильтрация (по названию, категории, бренду, цене)
- Валидация данных при добавлении

### Справочники

- Управление категориями и брендами
- Защита от удаления используемых категорий/брендов
- Проверка уникальности названий

### Сохранение данных

- Персистентное хранение данных
- Автоматические миграции через Liquibase
