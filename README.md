# Каталог Товаров (Multi-module Project)

## Описание проекта

Мультимодульное веб-приложение для управления каталогом товаров с системой аутентификации и авторизации, расширяемыми
модулями логирования и аудита.

## Структура проекта

Проект состоит из трех модулей:

### [app](app/) - Основное приложение

Веб-приложение для управления каталогом товаров.

- **Технологии**: Java 17, Spring Boot, PostgreSQL, Liquibase, Redis
- **Функциональность**:
    - Аутентификация и авторизация
    - CRUD операции с товарами, пользователями, категориями, брендами
    - Фильтрация товаров по различным критериям
    - Валидация данных
    - Кеширование с Redis

### [my-audit](my-audit/) - Модуль аудита действий

Модуль для отслеживания и записи действий пользователей в системе.

- **Технологии**: Spring AOP, SLF4J, Logback
- **Функциональность**:
    - Аудит входящих HTTP запросов
    - Логирование действий контроллеров
    - Асинхронная запись логов

### [my-logger](my-logger/) - Модуль детального логирования

Модуль для детального логирования выполнения бизнес-логики.

- **Технологии**: Spring AOP, SLF4J, кастомные аннотации
- **Функциональность**:
    - Логирование времени выполнения методов
    - Аннотация `@Loggable` для пометки логируемых методов / классов

## Запуск проекта

1. **Клонирование репозитория:**
   ```bash
   git clone <repository-url>
   cd <project-directory>
   ```
2. **Запуск инфраструктуры:**
    ```docker-compose -f docker/docker-compose.yml up -d```
3. **Сборка и запуск приложения:**
    ```bash
   # Сборка всех модулей
    mvn clean install

    # Запуск основного приложения
    cd app
    mvn spring-boot:run
   ```
4. **Приложение будет доступно по адресу: http://localhost:8080**
